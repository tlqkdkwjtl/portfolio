프로젝트

We Game Player
2025.11 ~ 2025.12
팀 프로젝트 · 2인 +
AI를 통해 게임 정보를 반자동으로 갱신하는 서비스
개요

게임별 필요 정보와 확률 계산이 복잡하고 이벤트마다 새로운 계산식이 필요한 문제를 해결하기 위해, AI를 통해 게임 정보를 반자동으로 갱신하는 서비스입니다.

역할 · 구현
  • 게임별 이력 테이블 설계 및 게임사 제공 API 연동
  • AI를 통한 반자동 정보처리 및 새로운 계산식 생산 구현
  • 파이프라인을 통한 데이터 분류 구현
  • Spring Security + JWT 기반 로그인 / 회원가입 API 설계 및 구현
  • 프로젝트 완성을 위한 주도적 일정 및 스케줄 관리
사용한 기술 스택
Spring Boot MySQL AWS EC2 GitHub Actions
배운 점

RESTful API 설계 원칙과 스프링 레이어드 아키텍처를 실전에서 적용하며 백엔드 구조를 깊이 이해하게 됨. 배포 자동화(CI/CD)를 처음 경험.

🔒 비공개
솟대쟁이
2025.11 ~ 현재
수업 / 개인 프로젝트 +
가상의 한국 도시를 운영하는 전략 시뮬레이션 게임
개요

기말 과제 프로젝트로 제출한 전략 시뮬레이션 게임으로, 가상의 한국 도시 관리자로서 예산을 재원 삼아 도시를 운영하는 게임입니다. 초기 버전은 HTML/CSS/JavaScript 기반으로 구현했으며, 현재는 Unity 환경으로 전환해 기능과 시스템을 확장 개발 중입니다.

역할 · 구현
  • 기획에 맞는 게임 UI 설계 및 실제 개발 구현
  • 기말 과제 프로젝트로 기한 내 개발 완료
  • 과제 제출 이후 부족한 부분 추가 개발 중
사용한 기술 스택
Unity C# JavaScript GitHub
배운 점 · 성과

기획부터 배포까지 혼자 경험하며 프로젝트 전 단계를 파악. 실제 게임 출시를 위한 저작물 등록 관련 경험.

around_you
2025.11 ~ 2025.12
수업 프로젝트 +
당근 앱에 없는 기능을 직접 기획하고 구현한 크로스플랫폼 프로젝트
개요

당근 앱 사용 중 추가되었으면 하는 기능을 직접 구현한 프로젝트. 당근 제공 기능 구현 + 추가 기능 개발.

역할 · 구현
  • 당근 앱 유사 환경 구현 — 모바일, PC 프로그램, 웹 3곳에서 동시 연동 가능한 프로그램 완성
  • AI를 통한 게시물 자동화 시스템 구현
  • 한국어 외 영어, 일본어, 중국어, 러시아어 등 다국어 번역 기능 구현
사용한 기술 스택
Flutter Android MySQL
배운 점

처음으로 PC, 웹, 모바일이라는 서로 다른 환경을 동시에 구현하며 크로스 플랫폼 개발의 복잡성과 연동 구조를 체득.
